Модуль complex_number предоставляет класс ComplexNumber для работы с комплексными числами и основные математические операции над ними. Комплексное число представляется в виде a + bi, где a - действительная часть, b - мнимая часть.

Основные возможности:
1.Арифметические операции (сложение, вычитание, умножение)

2.Вычисление модуля комплексного числа

3.Операция комплексного сопряжения

4.Поиск числа с наибольшим модулем в списке

5.Корректное строковое представление

Один из способов применения - это создание комплексных чисел:


# Создание комплексных чисел

from complex_number import ComplexNumber

num1 = ComplexNumber(3, 4)    # 3 + 4i
num2 = ComplexNumber(1, -2)   # 1 - 2i  
num3 = ComplexNumber(0, 5)    # 5i
num4 = ComplexNumber(2, 0)    # 2
